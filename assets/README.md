# Assets

This folder contains images and files referenced in the documentation.

## Structure

The assets folder is organized into subdirectories:

### images/

#### companies/
- Company logos and organizational materials
- Office locations in Rome
- Team and colleague photos
- Technology infrastructure diagrams

#### certifications/
- Professional certification scans
- Training certificates

#### logos/
- Company logos (Atos, TeleAp, RSI Sistemi, Avaya)
- Standards organization logos
- Vendor logos

### files/

#### publications/
- Technical documents
- Internal publications
- Industry whitepapers

#### certifications/
- Digital copies of certification documents
- Training certificates
- Professional credentials

All assets are referenced from markdown files using relative paths from the docs folder.
